Ressources

Cette page regroupe des liens vers les outils, les documents, le sites web que nous utilisons. Elle contient tout ce qu'il faut pour prolonger un atelier MixTeen chez soi.

Scratch

On ne présente plus Scratch, ce logiciel est utilisé dans les écoles et le collèges Français.

Il y a plusieurs versions :

  • Scratch 1.4 (obsolète) à installer
  • Scratch 2 en ligne ou à installer
  • Scratch 3 (béta) en ligne
Notre studio contient quelques réalisations.

Pour débuter, il existe de la documentation en français :

  • Des cartes et un guide
  • Un Livre Scratch pour les kids aux éditions Eyrolles
  • Le Code Club Scratch

Python

Python est un langage de programmation utilisé dans les plus grandes entreprise comme Google, Facebook... Pour l'installer, il suffit de se rendre sur le site officiel.

Nous utilisons des librairies graphiques comme Turtle ou PyGame Zero. Nous recommandons l'éditeur Mu car il est dédié aux enfants.

Pour débuter, il existe de la documentation en français :

  • Un livre Python pour les kids aux éditions Eyrolles
  • Le Code Club Python

Unplugged (sans ordinateur)

Il s'agit de pratiquer des activités sans ordinateur pour découvrir de façon ludique et en pratique les bases et les concepts de la programmation informatique. Nous avons notamment pratiqué avec les enfants : compter en binaire, décoder des messages secrets, dessiner des pixels, ou encore d’autres "tours de magie" qui cachent en réalité des principes informatiques ...

  • Computer Science Unplugged en Anglais est un livre élaboré par des enseignant et régulièrement mis à jour depuis 2005
  • Les enfants codaient

Thymio

Thymio est un petit robot doté d'une grande intelligence et qui sait faire plein de choses.

Il se programme avec visuellement Aseba Studio.

Micro:bit

Micro:bit est une petite carte programmable conçue à l'origine pour l'éducation britannique. Elle embarque des capteurs (température, boussole, gyroscope) et des connecteurs permettent toutes sortes de réalisations.

Micro:bit se programme :

  • Visuellement avec Makecode
  • Textuellement en Javascript ou en Python

Pour débuter, beaucoup de documentation reste en anglais, mais il existe de la documentation en français :

  • Leçons Micro:bit en Anglais
  • Documentation de référence en Anglais
  • Code Club Micro:bit

Nous utilisons des kits de prototypage électronique par Kitronik (Inventor's kit) et Protopic (Exhi:bit).

Le web, HTML, CSS & JavaScript

Créer son propre site web c'est facile!

Il s'agit découvrir le fonctionnement du web (HTTP, serveurs web et navigateurs), de créer une page en utilisant les balises HTML, de la mettre en forme (couleurs, polices...) avec CSS et de l'animer avec JavaScript.

Nous avons créé nos propres supports :

  1. Comment marche internet?
  2. Comment fonctionne le web?
  3. Construire une page web en HTML
  4. Programmer en JavaScript
