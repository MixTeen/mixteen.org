Accord sur l'utilisation d'images et de vidéos d'enfants

Lors d'un atelier MiXTeeN, nous prendrons des photos et des vidéos de vos enfants.

Nous les utiliserons pour :

  • Promouvoir cet événement dans le monde entier
  • Les mettre sur notre site web, dans des publications (en ligne et sur papier)
  • Les partager avec certains médias (les images pourraient apparaître dans les journaux ou être diffusées sur les chaînes télévisées)

Conditions d’utilisation :

  • Nous n’utiliserons pas de données personnelles ou de noms complets (c’est-à-dire le prénom et le nom) d’un enfant ou d’un adulte dans une photo ou vidéo, sur notre site web, dans nos brochures ou dans nos autres publications.
  • Nous ne mettrons pas d’adresses mail privées ou postales, ni de numéros de téléphone ou de fax sur des vidéos, sur notre site web, dans nos prospectus ou dans toutes autres publications.
  • Si nous utilisons une photo individuelle d’un enfant, nous n’utiliserons pas le nom de cet enfant dans la légende de cette photo.
  • Si nous citons le prénom d’un enfant dans un texte, nous n’utiliserons pas de photo de cet enfant pour illustrer cet article.
  • Nous utiliserons uniquement des images d’élèves correctement habillés, afin d’éviter tout risque d’utilisation inappropriée de ces images.
